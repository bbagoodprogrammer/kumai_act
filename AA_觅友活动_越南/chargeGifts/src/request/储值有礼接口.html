2.领取奖励

http://activity.udateapp.com/index.php?action=chargeGifts.get&amount={amount}&signture=innerserver&uid={uid}&token={token}

参数

  • amount 门槛金币值

code

  • 30001 任務未達成
  • 30002 已經領取過了
{
  "response_status": {
      "error": "",
      "code": 0
  },
  "response_data": {
      "prizes": [ // 获得奖励（结构与init接口一致）
          {
              "id": 1,
              "name": "鼓掌",
              "image": "http://udatefile.cat1314.com/uc/img/image_1568887570.png?wsSecret=2c0433064e5a7a04ca4599417a73ab7a&wsTime=1599062400",
              "price": 10,
              "type": "gift",
              "count": 1
          }
      ]
  }
}
